# K-blog

基于nodejs搭建的轻微博

## 介绍
K-blog 是使用 **Node.js** 及 **MongoDB**数据库, **Express**框架, **ejs**模板, 搭建的一个学习型的轻微博,
功能较简单, 目前正在研究建设中, 有不对的地方还望大家提出修正。

##安装部署
```bash
//install node npm mongodb
//run mongodb
$ cd k-blog
$ npm install
$ node app.js
```

##其它
 **K-blog** 采用 nodejs v0.8.4 版本.

如: 启动或访问时报一些奇怪错误,可检查 **package.json** 文件各组件是否存在版本兼容问题.
